### 嵌入式设备
- 安装包 ipk
- 包管理工具 opkg
```bash
opkg update # 更新可以获取的软件包列表
opkg upgrade # 对已经安装的软件包升级
opkg list # 获取软件列表
opkg install # 安装指定的软件包
opkg remove # 卸载已经安装的指定的软件包

# 打印系统架构
opkg print-architecture
```
