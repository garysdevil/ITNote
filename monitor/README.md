
# 监控系统
## 监控划分
参考文档
https://my.oschina.net/u/4278828/blog/3200478/print

1. 针对不同场景把监控系统分为三类
    - 日志类 Log
    - 调用链类 全链路监控
    - 度量类 Metric

2. 监控系统的分层
    1. 客户端监控
    2. 业务层监控
    3. 应用层监控
    4. 系统层监控
        - 内存
        - CPU
        - 存储：硬盘掉线 硬盘读写错误 读写超时
    5. 网络层监控
        - 流量速率
    6. 硬件层

3. 安全监控
