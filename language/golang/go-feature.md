# Go
- 将并发机制纳入Go的关键字
- 编译为二进制但又带有垃圾回收机制和运行时反射能力

## 特性1
1. 自动垃圾回收

2. 函数多返回值

3. 异常处理
传统OOP使用try（）catch{}模块捕获异常，Go引入三个关键字defer、panic和recover

3. 匿名函数和闭包

4. 类型和接口

5. 高并发

6. 支持反射

7. 语言交互性
重用了大部分的C模块，Cgo允许开发者混合编写C语言代码
import "C"

## 特性2
1. Go语言追求显式的表达，避免隐含。 
    1. 禁止参数默认值

2. 强类型语言
